# Classic-Voice-Panel 2020

Recreation of the classic 2020 Discord voice panel.

*Note: Requires Vencord or BetterDiscord to function. Using the "IconRevert 2023" and "Oldcord" plugins is highly recommended for the best visual results.*

## Installation

### Local method
1. Download [Classic-Voice-Panel 2020.css](https://raw.githubusercontent.com/jm1999-d/Classic-Voice-Panel-2020/refs/heads/main/Classic-Voice-Panel%202020.css) (right-click > "Save As").
2. Move the file to the [Vencord](https://vencord.dev/) or [BetterDiscord](https://betterdiscord.app/) theme folder:
   - `Settings` > `Themes` > `Open theme folder`

### Online method
Paste the following URL in `Settings` > `Themes` :
- `https://raw.githubusercontent.com/jm1999-d/Classic-Voice-Panel-2020/refs/heads/main/Classic-Voice-Panel%202020.css`
